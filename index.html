Yahoo Crawler Archive

Spring-Fall 2021

Personal Project

The Yahoo Crawler project is both a web crawler and search engine for the now defunct Yahoo Answers website. This was a "joke" project that started with a friend asking random questions from the site. Questions like "Will my laptop get heavier if I put more files on it?". It was good motivation to get at the very least a prototype working.

Programmed with Javascript & Python. Designed with XAMPP, JSON, and HTML.

  • Codeveloped the web crawler using
  • Created a local MySQL database for the questions, answers, and various users. The database was made with along with Python and XAMPP
  • Converted MySQL tables to JS files for local storage within the Github website
  • Built on top of an already existing frontend website & connect to locally stored data

Combat Evolved

Winter 2020

VGDC Project

Combat Evolved is a 2-4 player competitive top down tank shooter. The goal, destory everyone else's tanks. With various levels, weapons, and power ups, Combat Evolved is a quick and easy to pick up game for anyone.

Programmed w/ C# & Unity. Released on Windows.

  • Created the local multiplayer framework using Unity's experimental Input System.
  • Implemented a camera system where it zooms based on player's positions
  • Designed a variety of the player's weapons and abilities with a generic base class
  • Created environmental hazards/mechanics for the player to interact.
  • Added special effects using Unity's particle effects.
